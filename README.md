## Editing this site

Everything lives in `index.html`. There's no compiling or bundling required: open the file in any code editor, make a change, save it, and push to this repository. GitHub Pages will rebuild automatically within a minute or two of the push.

If you're forking this for your own portfolio, the color palette and layout are controlled by the CSS variables near the top of the `<style>` block, so you can reskin the whole site by changing a handful of values there.
